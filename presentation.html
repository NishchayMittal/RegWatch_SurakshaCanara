Agent Deep-Dive 1/3

Watcher & Dedup Agent

👁️

Watcher Agent

📂

Multi-Source Ingestion

Scans local JSON/PDF directories, RSS feeds, and regulator websites

📄

PDF Text Extraction

Uses PDFPlumber for robust text extraction from circular documents

🏷️

Structured Output

Produces CircularDoc objects with URL, title, regulator & raw text

🔍

Dedup Agent

🧬

Semantic Deduplication

Embeds circulars using all-MiniLM-L6-v2 sentence-transformers

📐

Cosine Similarity

Computes pairwise similarity scores; threshold-based duplicate flagging

🚫

Prevents Redundancy

Catches semantically equivalent circulars even with different wording

Agent Deep-Dive 2/3

MAP Extractor & Router Agent

📝

MAP Extractor

🔤

Rule-Based NLP

Detects "shall", "must", "mandated", "directed" keywords to extract Mandatory Action Points

📊

Confidence Scoring

Each MAP gets a score; overall < 0.85 triggers Human Review Queue

🔌

LLM-Ready

Ollama integration available for local generative LLM extraction if GPU available

🧭

Router Agent (RAG)

🗄️

ChromaDB Vector Store

Knowledge base of 9 departments embedded for semantic retrieval

🎯

Semantic Matching

Each MAP is matched to the closest department via embedding similarity

⏱️

SLA Assignment

Assigns deadlines (5–21 days) based on regulator-specific SLA rules in the database

Agent Deep-Dive 3/3

Notifier & Validator Agent

🔔

Notifier Agent

📧

Multi-Channel Alerts

Dispatches simulated Email & Slack notifications to department queues

🚨

Re-Escalation Alerts

📋

Full Audit Trail

✅

🔎

Evidence Validation

Why RegWatch?

Key Differentiators

🔒

100% Offline — Zero API Dependency

Runs entirely on local HuggingFace models + ChromaDB. No internet needed after setup. Ideal for air-gapped banking environments.

👤

Human-in-the-Loop Safety Net

Low-confidence extractions (<0.85) are automatically routed to a Human Review Queue before being assigned.

🔁

Closed-Loop Validation & Re-Escalation

Validator Agent checks evidence completeness. Failures trigger automatic re-escalation alerts — no task falls through.

🧬

Semantic Deduplication

Catches duplicate circulars even with different wording using cosine similarity on sentence embeddings.

📝

Full Audit Trail

Every action — ingestion, routing, validation, deletion — is logged with local timestamps for regulatory accountability.
